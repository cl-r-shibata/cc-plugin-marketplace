# Greeting Plugin

A simple Claude Code plugin that provides Japanese greetings.

## Features

- `/greeting:hello` - Say hello in Japanese (こんにちは - Konnichiwa)

## Installation

```bash
cc plugins install <marketplace-url>
```

Or manually add to your Claude Code plugins directory.

## Usage

Simply run:
```
/greeting:hello
```

Output: "こんにちは (Konnichiwa) - Hello"
